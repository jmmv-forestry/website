--- layout: post title: "GNOME 2.6.2 released" date: 2004-06-28 17:15:00 -0400 categories: julipedia: 2004/06/gnome-262-released.html excerpt: Post imported from The Julipedia; excerpt not available. --- The GNOME Project has just released the 2.6.2 version of its desktop environment, probably the last release of the 2.6 stable branch. The official announcement provides more information, including a list of changes since the previous version.

Furthermore, I've just updated all the GNOME packages in pkgsrc to their respective versions in this new release. That is, you can easily install GNOME 2.6.2 with the simple cd pkgsrc/meta-pkgs/gnome && make install command... and on multiple platforms! This may be a good reason to try this excellent package system today ;-)
